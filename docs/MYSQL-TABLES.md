# MySQL data model (26 tables)

Authoritative schema: `prisma/schema.prisma`. The canonical SQL is generated by
`prisma migrate dev` into `prisma/migrations/*/migration.sql`. A hand-written
reference DDL is in `docs/mysql-schema.reference.sql`.

Conventions: UUID ids are `CHAR(36)`; Auth.js ids are `cuid` strings; timestamps
are `DATETIME(3)` in **UTC**; `jsonb`→`JSON`; `numeric`→`DECIMAL`; PostgreSQL
enums→native enum-equivalent columns validated by Prisma enums; RLS→enforced in
application code (`src/lib/auth`, repositories, server actions).

## Authentication (Auth.js / NextAuth v5)

### 1. users
Identity table other tables link to (replaces Supabase `auth.users`).
Columns: `id` cuid **PK**, `name?`, `email?` **unique**, `emailVerified DATETIME(3)?`, `image? TEXT`, `createdAt`. Related: reviewer profiles, admins, moderation/audit actors.

### 2. accounts
OAuth/link accounts for a user. `id` **PK**; FK `userId`→users **ON DELETE CASCADE**; **unique**(`provider`,`providerAccountId`); index(`userId`). Token columns are `TEXT`.

### 3. sessions
Database sessions. `id` **PK**; `sessionToken` **unique**; FK `userId`→users **CASCADE**; `expires`.

### 4. verification_tokens
Magic-link tokens. `token` **unique**; **unique**(`identifier`,`token`); `expires` (enforces link expiry).

## Review platform

### 5. programs
Verified Portfolio Builders offerings. `slug` **PK**; `title`, `type` (ExperienceType), `officialUrl? TEXT`, `summary TEXT`, `seoIntro TEXT`, `sortOrder`, `active`, `createdAt`. Children: reviews. Feature: program review pages, sitemap, filters.

### 6. review_categories
Rating dimensions. `key` **PK**; `label`, `description TEXT`, `sortOrder`, `active`. Children: category_ratings. Feature: category scores, write-review form.

### 7. reviewer_profiles
Optional reviewer identity. `id` uuid **PK**; `userId?` **unique** FK→users **ON DELETE SET NULL**; `displayName`; `emailHash` **unique**; `createdAt`. Feature: account page (a user's own submissions).

### 8. reviews
First-party reviews. `id` uuid **PK**; `slug` **unique**; `experienceType`; `programSlug?` FK→programs **SET NULL**; `programScope` (non-null normalized dedupe key, = programSlug or ""); `title`, `body TEXT`, `overallRating`; `categoryRatings/pros/improvements JSON`; `outcome? TEXT`, `wouldRecommend?`; `reviewerDisplayName`, `reviewerEmailHash` (never public), `reviewerId?` FK→reviewer_profiles **SET NULL**; `relationship`, `batch?`, `experienceDate DATE`, `verified`, `status` (ReviewStatus), `helpfulCount`, `reportCount`, `spamFlags JSON`, `isSample`, and UTC timestamps (`submittedAt`, `verifiedAt?`, `publishedAt?`, `lastEditedAt?`, `moderatedAt?`, `removedAt?`).
- **Duplicate prevention:** unique(`reviewerEmailHash`,`experienceType`,`programScope`) — works even when `programSlug` is NULL because `programScope` is non-null.
- Indexes: (`status`,`publishedAt`), (`programSlug`,`status`), (`experienceType`,`status`).
- Statuses: draft, pending_verification, pending_moderation, approved, rejected, flagged, removed. Only `approved` is public/counts toward ratings.

### 9. category_ratings
Per-dimension scores. `id` uuid **PK**; FK `reviewId`→reviews **CASCADE**; FK `categoryKey`→review_categories **CASCADE**; `value`; **unique**(`reviewId`,`categoryKey`); index(`categoryKey`).

### 10. verification_records
Email-verification trail. `id` uuid **PK**; FK `reviewId`→reviews **CASCADE**; `method`, `tokenHash?`, `verified`, `createdAt`, `verifiedAt?`; index(`reviewId`).

### 11. company_responses
One official reply per review. `id` uuid **PK**; FK `reviewId`→reviews **CASCADE**, **unique**; `authorName` default "Portfolio Builders"; `body TEXT`; `respondedAt`. Never changes ratings.

### 12. helpful_votes
Abuse-resistant votes. `id` uuid **PK**; FK `reviewId`→reviews **CASCADE**; `voterHash`; **unique**(`reviewId`,`voterHash`). One vote per fingerprint; created + counter incremented in one transaction.

### 13. review_reports
Abuse/accuracy reports. `id` uuid **PK**; FK `reviewId`→reviews **CASCADE** (route resolves slug→id); `reporterHash`; `reason`, `details TEXT`, `status`; **unique**(`reviewId`,`reporterHash`) prevents uncontrolled duplicates; index(`status`). `reportCount` incremented atomically.

### 14. moderation_actions
Moderation trail. `id` uuid **PK**; FK `reviewId`→reviews **CASCADE**; FK `actorId?`→users **SET NULL**; `action`, `reason? TEXT`, `createdAt`; index(`reviewId`).

### 15. review_attachments
Attachment **metadata only** (files live in S3/Cloudinary/Vercel Blob). `id` uuid **PK**; FK `reviewId`→reviews **CASCADE**; `storagePath TEXT`, `mimeType`, `sizeBytes`, `createdAt`; index(`reviewId`).

## External review sources

### 16. review_sources
Source catalogue. `slug` **PK**; `name`, `sourceGroup`, `ratingType`, `officialUrl? TEXT`, `defaultMin/Max DECIMAL(6,2)`, `integrationMode`, `countsTowardLearnerScore`, `attributionRequirements? TEXT`, `complianceNotes? TEXT`, `syncEnabled`, `active`, `updatedAt`. Children: profiles, external reviews, sync jobs, import batches, credential.

### 17. external_profiles
A verified profile on a platform. `id` uuid **PK**; FK `sourceSlug`→review_sources **CASCADE**; `externalProfileId?`; `externalProfileUrl TEXT`; `externalBusinessName`; `integrationMode`; `verificationStatus`; FK `verifiedById?`→users **SET NULL**; ratings/counts (`DECIMAL`), timestamps; `active`, `isSample`. **unique**(`sourceSlug`,`externalProfileId`); index(`active`,`verificationStatus`). Only active+verified shown publicly.

### 18. external_reviews
Imported external reviews. `id` uuid **PK**; FK `sourceSlug`→review_sources **CASCADE**; FK `externalProfileId`→external_profiles **CASCADE**; author/body/scale fields; `originalRating/Min/Max DECIMAL`, `originalRatingLabel`, `normalizedRating? DECIMAL(4,2)` (NULL when not star-compatible); `sourceGroup`; dates; `verification`, `attribution TEXT`, `contentHash`, `visibility`, `removed`, `isSample`. **unique**(`sourceSlug`,`externalProfileId`,`externalReviewId`) and (…,`contentHash`); indexes on (`sourceSlug`,`visibility`,`removed`) and (`externalProfileId`).

### 19. sync_jobs
Sync run log. `id` uuid **PK**; FK `sourceSlug`→review_sources **CASCADE**; FK `externalProfileId?`→external_profiles **CASCADE**; `jobType`, `status`, counters, `errorSummary? TEXT`, timestamps; index(`sourceSlug`,`startedAt`).

### 20. import_batches
CSV/import batches. `id` uuid **PK**; FK `sourceSlug`→review_sources **CASCADE**; FK `adminId?`→users **SET NULL**; `format`, `status`, `totals JSON`, `createdAt`; index(`sourceSlug`).

### 21. source_credentials
Credential **references only** (prefer env/secret manager). `sourceSlug` **PK** FK→review_sources **CASCADE**; `credentialRef?`; `encryptedSecret BYTES?` (only if you must; never plaintext); FK `updatedById?`→users **SET NULL**; `updatedAt`. Never selected by public repositories.

## Administration & website

### 22. admin_users
Staff roles. `userId` **PK** FK→users **CASCADE**; `role` (moderator|admin); `createdAt`. Drives all authorization.

### 23. audit_logs
Immutable audit trail. `id` uuid **PK**; FK `actorId?`→users **SET NULL**; `action`, `targetId?`, `detail? TEXT`, `createdAt`; index(`createdAt`). Secrets are never written.

### 24. contact_messages
Contact form. `id` uuid **PK**; `name`, `email`, `topic`, `message TEXT`, `createdAt`; index(`createdAt`).

### 25. site_settings
Key/value settings. `key` **PK**; `value JSON`.

### 26. seo_settings
Per-path SEO overrides. `path` **PK**; `title?`, `description? TEXT`, `ogImage? TEXT`, `noindex`.

### (+) aeo_questions
Managed direct-answer content. `id` uuid **PK**; `question TEXT`, `answer TEXT`, `pagePath`, `sortOrder`, `active`, `updatedAt`; index(`pagePath`,`active`).

> Note: the brief lists 26 numbered business/auth/website tables plus `aeo_questions`; all are present. `authenticators` (WebAuthn) is intentionally **omitted** because passkeys are not implemented.
